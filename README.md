# React Native Google UMP

React Native bindings for Google's User Messaging Platform SDK

## Installation

```sh
npm install react-native-google-ump
```

## Usage

```js
import { multiply } from "react-native-google-ump";

// ...

const result = await multiply(3, 7);
```
